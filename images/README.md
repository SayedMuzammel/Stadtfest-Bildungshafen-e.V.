# Bilder-Ordner

Lege hier die Fotos der Speisen ab. Solange ein Foto fehlt, zeigt die Seite
automatisch einen schönen, farbigen Platzhalter mit „Foto folgt“ – es ist also
nichts kaputt, wenn ein Bild noch fehlt.

## Empfehlungen
- Format: **JPG** (oder WebP), quer/landscape, ca. **800 × 550 px**
- Pro Datei möglichst **unter 300 KB** halten (schnelles Laden auf dem Handy)
- Dateinamen **klein** schreiben, ohne Leerzeichen/Umlaute (siehe Liste unten)
